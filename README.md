# deep-research-pro — Multi-source research agent with cited reports

> Breaks any research topic into sub-questions, searches multiple sources, synthesizes findings, and delivers a structured cited report. No paid APIs required.

[![OpenClaw Skill](https://img.shields.io/badge/OpenClaw-Skill-blueviolet)](https://github.com/NachaFromMars)

## Overview
deep-research-pro is a self-contained research workflow that produces thorough, cited reports from multiple web sources. It decomposes your topic into 3–5 focused sub-questions, executes targeted DuckDuckGo searches (web + news) for each, synthesizes the results, and delivers a structured report with source citations. No paid search APIs needed — it runs entirely on DuckDuckGo.

## Features
**5-step workflow:**
1. Clarify goal (1–2 questions)
2. Plan 3–5 research sub-questions
3. Execute multi-source search (DDG web + news per sub-question)
4. Synthesize with inline citations
5. Deliver structured report

**Report format:**
- Executive Summary
- Findings by sub-question (cited)
- Gaps & Uncertainties
- Recommendations

## Usage / Quick Start
Trigger with a research request. The skill asks 1–2 quick clarifying questions, then runs the full pipeline autonomously.
